#### INSTALLATION

1. Install a virtual environment
    ```sh
    $ virtualenv -p python3.5 env
    ```

2. Install all packages in the `requirements.txt` file.
    ```sh
    $ env/bin/pip install -r requirements.txt
    ```

3. Now you can run
    ```sh
    $ env/bin/python run.py
    ```
